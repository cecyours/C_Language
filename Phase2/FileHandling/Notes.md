
> File : File is a collection of Fact with extension like .pdf , .md, .jpg, .pm3, .c, .cpp etc

## Modes of File 

r : read  -> Return NULL if File Doesn't exists

w : write   ->  create the file If Don't exists. 

a : append -> append the data to end of file..

--------------

r+: read/write

w+: write/read 

a+: append/read
